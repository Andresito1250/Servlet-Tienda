# Servlet-Tienda

Web application developed with Java Servlets and MySQL that simulates an online clothing store. The project allows user registration, product purchasing, account management, and CRUD operations connected to a relational database.

## Features

- User registration and authentication
- Product catalog visualization
- Purchase processing system
- CRUD operations for user accounts
- MySQL database integration
- Session handling with Servlets
- Responsive HTML/CSS interface
- JDBC database connection

## Technologies Used

- Java
- Java Servlets
- HTML5
- CSS3
- MySQL
- JDBC
- Apache Tomcat
- NetBeans

## Database Structure

The application uses a MySQL database with the following tables:

- `usuarios1`
- `productos`
- `compras`

## Project Architecture

The system follows a client-server architecture using Java Servlets for backend request handling and HTML/CSS for frontend visualization.

## Main Modules

### User Registration
Allows users to create an account and stores information securely in the database.

### Product Catalog
Displays available products including clothing items and allows purchases.

### Purchase System
Processes purchases and stores transaction records in the database.

### User CRUD
Users can update or delete their accounts dynamically.
